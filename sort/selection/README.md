# 選擇排序

將陣列區分成未排序與已排序兩這陣列，然後從未陣列中找出最小(大)的值，並且把該值加入倒已經排序中陣列最後。最後遍歷過所有未排序陣列就可以完成排序。

```
# 原始陣列，最小值: 0
[8, 4, 3, 0]  

# 第一輪
## 已排序
[0]
## 未排序，最小值: 3
[8, 4, 3]

# 第二輪
## 已排序
[0, 3]
## 未排序，最小值: 4
[8, 4]

# 第三輪 
## 已排序
[0, 3, 4]
## 未排序，最小值: 8
[8]

# 第三輪
## 已排序
[0, 3, 4, 8]
```

透過雙層迴圈方式分別找出每一輪陣列中的最小值所在位置

```go
for i := 0; i < len(data)-1; i++ {
minIndex := i // 紀錄陣列中最小值索引
for j := i + 1; j < len(data); j++ {
if data[minIndex] > data[j] {
minIndex = j
}
}
}
```

時間複雜度: $O(n^2$)
