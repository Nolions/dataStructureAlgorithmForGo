# 插入排序

從陣列中取得i值，並且與i-1值進行排序，如果該i值較小則往前移，並繼續與下一位置值近進行比較，但如果i值較大，則進到下一輪，並重複之前的步驟。

```
# 原始陣列
[8, 4, 3, 0]

# 第一輪，i值:4，並且與之前的值逐一比對，如果i值較小則往前移一位
[4, 8, 3, 0] // 4 < 8，4往前移

# 第二輪，i:2、值:3
[4, 3, 8, 0] // 3 < 8，3往前移
[3, 4, 8, 0] // 3 < 4，3往前移

# 第三輪，i:2、值:0
[3, 4, 0, 8] // 0 < 8，0往前移
[3, 0, 4, 8] // 0 < 4，0往前移
[0, 3, 4, 8] // 0 < 3，0往前移
```

時間複雜度: $O(n^2$)
